To install from github (using ssh)
```
git clone git@github.com:mattJohnston1/my-app.git
cd my-app
```


To run backend (port is on 8080):
```
cd server/
npm install
npm start
```


To run frontend (port is on 3000):
```
cd simple-web-app/
npm install
npm start
```


To access web app go to http://localhost:3000
